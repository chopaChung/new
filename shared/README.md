# shared

이 디렉토리는 애플리케이션 전반에서 사용되는 **재사용 가능한 아토믹(Atomic) UI 컴포넌트**를 관리합니다.

이곳의 컴포넌트는 **비즈니스 로직(도메인 맥락)을 전혀 알지 못하며**, 

오직 `props`를 통해 전달받은 데이터와 스타일만 렌더링하는 Pure Component여야 합니다.

<br>

## 핵심 원칙 

1.  **도메인 의존성 제거 (Stateless)**
    * `auth`, `segment` 등 특정 비즈니스 로직에 의존하지 않습니다.
    * API 호출이나 전역 상태(Store) 구독을 하지 않습니다.
2.  **높은 재사용성 (Reusability)**
    * 어떤 페이지, 어떤 프로젝트에 가져다 놓아도 디자인 시스템에 맞게 동작해야 합니다.
3.  **스타일 중심 (Style-Driven)**
    * 디자인 시스템의 가이드라인을 준수하여 일관된 UI를 제공합니다.

<br>

## 📂 폴더 구조 

UI 요소의 종류에 따라 분류합니다.

```text
shared/
├── ui/                     # 버튼, 인풋, 카드 등 기본 요소
│   ├── Button.tsx
│   ├── Input.tsx
│   └── Modal.tsx

│   └── MainLayout.tsx
└── icons/                  # SVG 아이콘 컴포넌트
